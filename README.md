Project Overview

This Digital Literacy project was completed as part of the CSE0001 course. The main aim of this project is to help students understand how to use digital tools safely, communicate professionally online, and build a strong digital presence. In today's world, digital skills are very important for academic success and future career opportunities.

In this project, I completed five tasks related to digital literacy. These tasks helped me learn how to create digital content, set up professional online profiles, practice coding, write professional emails, and understand cybercrime awareness. All the work for this project is organized in this GitHub repository.
Module / Task Summaries
Task 1 – Digital Literacy Awareness Infographic

In this task, I created an infographic using Canva to explain the meaning and importance of digital literacy. The infographic includes topics such as safe internet practices, useful digital tools, and responsible online behavior. This task helped me improve my creativity and presentation skills.

Tool Used: Canva Output: Infographic screenshot saved in task-1-presentation/
Task 2 – Student Digital Portfolio

In this task, I created professional accounts on platforms like GitHub, LinkedIn, and HackerRank. These platforms are useful for showcasing skills, networking, and preparing for internships and jobs. Creating these profiles helped me start building my professional identity.

Platforms Used: GitHub, LinkedIn, HackerRank Output: Profile screenshots saved in task-2-portfolio/
Task 3 – Coding and Collaboration Platforms

In this task, I practiced coding by solving a beginner-level problem on HackerRank. I also created a Google Form called "Digital Literacy Awareness Quiz" with five questions related to online safety and digital skills. This task helped me understand how coding platforms and collaboration tools support learning.

Platforms Used: HackerRank, Google Forms Output: Screenshots saved in task-3-platforms/
Task 4 – Professional Email and Social Media Etiquette

In this task, I wrote two professional emails. One email was to a professor requesting an assignment extension, and the second email was to an internship coordinator expressing interest in an internship opportunity. I also created a checklist of social media Do's and Don'ts to promote responsible online behavior.

Output: Email drafts and checklist saved in task-4-email-etiquette/
Task 5 – Cybercrime Awareness Case Study and Prevention Guide

In this task, I researched a cybercrime topic called phishing. I wrote a case study explaining how phishing attacks happen and created a prevention checklist to help students stay safe online. This task increased my awareness of online security and cyber safety.

Output: Case study and prevention checklist saved in task-5-cybercrime/
